# Flutter To-Do App

A **beautiful Flutter To-Do app** with
- Animated task list (slide in/out)  
- Animated checkboxes  
- Delete tasks  
- Persistent storage with `SharedPreferences`

## Features

- Add, delete, and mark tasks as complete  
- Smooth animations for task addition/removal  
- Modern Figma-style UI  
- Data saved even after closing the app  

## How to Run

```bash
flutter pub get
flutter run
